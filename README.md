# ShulkerStacking-OG
Allow empty shulker boxes to stack like normal items. Not currently complete, but can be used if desired.

Current Target: Purpur 1.19.4

Authors: Barny1094875.

<h3>To build:</h3>

./gradlew build

The resulting .jar file will be in build/libs
